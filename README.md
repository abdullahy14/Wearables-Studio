# Wearables Studio

A premium custom apparel web application built with Next.js App Router. The platform includes:

- Luxury storefront and creator marketplace
- Interactive design editor with drag-and-position layers
- QR-powered checkout with **in-store payment only**
- Admin cashier dashboard for manual payment confirmation and production lifecycle control
- Role-based authentication for customers, creators, and admins

## Run locally

```bash
npm install
npm run dev
```

> Note: Orders are intentionally created as **Pending Payment**. No online payment is implemented.
