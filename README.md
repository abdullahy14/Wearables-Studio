# Wearables Studio

Wearables Studio is a production-minded full-stack web application for a premium custom apparel brand with a strict **in-store payment** workflow.

## Highlights

- Design studio with image upload, text composition, positioning controls, and T-shirt mockup preview.
- Creator marketplace with commissions tracked at **50 EGP** per completed sale.
- Product pages and checkout flow with **no online payment collection**.
- QR generation for designs, orders, and pickup retrieval.
- Admin cashier dashboard for payment confirmation, cancellations, design edit requests, and production status progression.
- Secure email/password authentication with signed HTTP-only session cookies and PBKDF2 password hashing.
